# Excel Sheet Column Number

Given a column title as appear in an Excel sheet, return its corresponding column number.

For example:

    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28 
    ...
Example 1:

Input: "A"<br/>
Output: 1<br/>
Example 2:<br/>

Input: "AB"<br/>
Output: 28<br/>
Example 3:<br/>

Input: "ZY"<br/>
Output: 701<br/>
 

Constraints:

1 <= s.length <= 7<br/>
s consists only of uppercase English letters.<br/>
s is between "A" and "FXSHRXW".<br/>
